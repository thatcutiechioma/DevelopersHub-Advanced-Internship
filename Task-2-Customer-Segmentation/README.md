# Task 2: Customer Segmentation Using Unsupervised Learning

## Objective
Cluster customers based on spending habits and propose marketing strategies for each customer segment.

## Dataset
Mall Customers Dataset

## Approach
- Loaded and explored the dataset
- Performed EDA (charts/plots)
- Applied K-Means clustering
- Used the Elbow Method to choose the number of clusters
- Visualized clusters using PCA
- Suggested marketing strategies for each cluster
